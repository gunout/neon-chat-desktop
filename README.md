# neon-chat-desktop
Neon Chat Desktop - Example ( not connected to the server ) 

# INSTALL 

    npm install
    npm start 
    npm run build

# INSTALL .DEB GENERER

    sudo dpkg -i dist/neon-chat-desktop_1.0.0_amd64.deb

# RUN

    /opt/NEON-CHAT/neon-chat-desktop
